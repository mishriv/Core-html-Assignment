HTML Tags

  1. < html>
    • Contain HTML elements on the page.
    • The root element of an HTML document.
  2. <head>
    • Contain meta information about the HTML document, like title and links to external resources.
    • Not displayed on the web age itself.
  3. <title>
    • Set the title of the web page, which displayed in the browser's title tab.
    • Essential for seo and user experience.
  4. <meta>
    • Provide metadata about the HTML dociment.
    • Important for SEO and rendering on various devices.
  5. <body>
    • Contain the visible contant of the web page.
    • User interactions and display elements go here.
  6. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
    • Heading of different levels, where h1 is the highest and h6 is the lowest.
    • Used for structing contant and indicating hierarchy
  7. <p>
    • Represents a paragraph of a text.
    • Used for organizing and formatting text contant.
  8. <a>
    • Defines a hyperlink to another web page or resource.
    • Essential for linking and navigation within a website.
  9. <img>
    • Embeds and image in the web page
    • Requires the src attribute to specify the image files URL.
  10. <ul>
    • Creates an unordered list(bulleted).
    • Used to group and display items without a specific order.
  11. <ol>
    • Creates an ordered list(numbered).
    • Used to display item in specific sequence.
  12. <div>
    • Used for grouping and stling content.
  13. <span>
    • Used for styling small portions of text or otherninline elements.
  14. <table>
    • Defines an interactive form for user input.
    • Contains variour form of controls.
